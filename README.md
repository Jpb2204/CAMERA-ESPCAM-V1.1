# Projeto Individual 01

*   Projeto desenvolvido para captura e armazenamento de imagens utilizando a placa ESP32-CAM.
*   Permite a captura de fotografias por meio de um botao fisico, exibicao de status em display LCD e armazenamento das imagens em cartao microSD.
*   As bibliotecas utilizadas para camera e cartao microSD podem ser consultadas na pasta components. Biblioteca usada: (https://github.com/espressif/esp32-camera)
*   A pasta Recursos possui imagens (Pinos do LCD, ESP32-CAM e do botao) e demais documentos de apoio (links uteis) para ajudar no desenvolvimento do projeto.
